# CTBE course offerings
-This is a mobile application for my university project. It provides a guide for courses given at Addis Ababa University, CTBE. It's built using Jetpack compose. 
-To find the mobile app design, please visit this website:
https://app.visily.ai/projects/96be100e-41b6-4d0f-823a-e54260c25645/boards/1874457

* Features
- Course browsing
- Department categorization
- Modern Jetpack Compose UI
- Responsive layouts
- Navigation support
- Clean architecture

* Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio
- Material Design

* Future Improvements
- Firebase integration
- Authentication
- Search functionality
- Offline support
